== Sistem online de gestiune a burselor

Ingineria Programarii: 2014 - 2015

Aplicatia isi propune sa realizeze un sistem de gestiune a burselor acordate de catre facultate.  Utilizatorii aplicatiei se impart in trei categorii: studenti, manageri si administratori. Bursele se impart in: burse de performanta, burse de merit/studiu, burse sociale si burse sociale ocazioanale.
